Summary: Latitude vs. X

The purpose of this project was to analize how weather changes as you get closer to the equator. Toaccomplish this analysis, we first pulled data from the OpenWeatherMap API to aassemble a dataset on over 500 cities.

After assembling the dataset, we used Matplolib to plot various aspects of the weather vs. latitud. Factors we looked at included: temperature, cloudiness, wind speed, and humidity.This site provides the source data and visualization created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed
